📚 App de Vocabulário Erudito
Um sistema robusto de gerenciamento de vocabulário multilingue, desenvolvido para ajudar usuários a expandirem seus conhecimentos linguísticos com termos precisos e literários. O projeto utiliza Django como framework principal e MySQL para armazenamento de dados em larga escala.

🚀 Funcionalidades
Gestão Multilingue: Suporte para múltiplos idiomas (Português, Inglês, Alemão, etc.) através de uma arquitetura escalável.

Carga de Dados Automatizada (ETL): Script Python personalizado para importação massiva de palavras a partir de arquivos .txt, utilizando lógica de proteção contra duplicatas (get_or_create).

Painel Administrativo: Interface intuitiva para curadoria, edição e exclusão de termos cadastrados.

Ambiente Seguro: Configuração de variáveis de ambiente com python-dotenv para proteção de credenciais de banco de dados.

🛠️ Tecnologias Utilizadas
Linguagem: Python 3.x

Framework Web: Django

Banco de Dados: MySQL

Conector de Banco: mysqlclient

Ambiente Virtual: venv

📋 Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina:

Python 3.x

MySQL Server
